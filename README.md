# 作成基準

## 【新ページ作成時の確認条項】

1. ページのタイトルは全て正しいか
2. contactページもつけたか
3. リンク指定はしたか
4. 検索jsonに追加したか
5. historyシートを確認したか
6. pagesシートのアイコンを更新したか
7. 関連ページや外部リンクが必要なページは追加したか
8. 関連ページや外部リンクの表示を更新したか
9. URLのパスは間違っていないか
10. 複垢の場合複垢一覧に追加したか
11. 関連情報の場合関連情報一覧に追加したか
12. カテゴリを設定したか
13. カテゴリ一覧に追加したか
14. ページ一覧に追加したか
15. 年表に追加したか
